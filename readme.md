## How to use
    1. Login Pusher Account
    2. Add 
       - `pusher-app-id`
       - `pusher-key-id`
       - `pusher-key-secret`
    3. Run command `go run main.go`
    4. Open url `localhost:8000` for index page
    5. Open url `localhost:8000/push` for pushing notification to index page
